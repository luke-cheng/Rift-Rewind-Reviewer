# Self Deploy

## Pre-requsition

- AWS account
- Github account

## Getting Your Riot API Key

1. Go to [Riot Games Developer Portal](https://developer.riotgames.com/)
2. Sign in with your Riot account
3. Navigate to "API Keys" section
4. Create a new API key or use an existing one
5. Copy the API key (it looks like: `RGAPI-xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`)

## Getting the code

1. clone this repo
2. open AWS Amplify via your broswer. Usually via https://us-east-1.console.aws.amazon.com/amplify/apps
3. Click create new app, import directly via github with the purple `create new app` on rop right
